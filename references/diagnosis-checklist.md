# 实体店常见问题诊断清单

---

## 一、客流类问题

### 症状：没客流

**诊断步骤**

1. 先看位置
   - 周边有没有同品类店铺？生意如何？
   - 店门口人流多少？什么时段人多？
   - 招牌是否醒目？晚上有灯光吗？

2. 再看产品
   - 产品是否有特色？能不能一句话说清楚卖点？
   - 定价是否符合周边消费水平？
   - 菜单/产品线是否过于复杂？

3. 最后看推广
   - 有没有做线上推广（抖音/大众点评）？
   - 有没有线下引流动作（发传单/活动）？
   - 有没有老客带新客的机制？

**常见原因**
- 选址失败（人流根本不到）
- 产品同质化（没有差异化）
- 没做推广（藏在深闺无人识）

### 症状：有客流但不下单

**诊断步骤**

1. 进店不购买
   - 产品定价是否超出心理预期？
   - 店内环境是否让顾客犹豫？
   - 服务是否跟得上？

2. 浏览后离开
   - 产品是否有吸引力？
   - 销售话术是否到位？
   - 有没有促单动作？

**常见原因**
- 价格与价值不匹配
- 产品展示不够吸引人
- 缺乏销售技巧

---

## 二、转化类问题

### 症状：客单价低

**诊断步骤**

1. 顾客只点单品
   - 有没有套餐设计？
   - 有没有加购推荐？
   - 有没有会员特权？

2. 顾客只买基础款
   - 高价产品是否有价值感？
   - 有没有试吃/试用体验？
   - 销售人员是否会推高价产品？

**常见原因**
- 产品组合设计不合理
- 缺乏加购引导
- 高价产品缺乏价值感知

### 症状：成交率低

**诊断步骤**

1. 进店顾客购买比例
   - 统计100个进店顾客，有多少下单？
   - 正常水平：30%-50%（餐饮）、10%-20%（零售）

2. 咨询顾客成交比例
   - 统计咨询顾客，有多少成交？
   - 正常水平：40%-60%

**常见原因**
- 产品与顾客需求不匹配
- 价格策略问题
- 销售话术不到位

---

## 三、复购类问题

### 症状：一次性顾客多

**诊断步骤**

1. 有没有会员体系？
   - 会员权益是否有吸引力？
   - 办卡门槛是否合理？
   - 会员服务是否到位？

2. 有没有复购促动？
   - 消费后有没有回访？
   - 有没有优惠券？
   - 有没有定期活动？

**常见原因**
- 产品不过硬（不想再来）
- 没有会员体系（没理由再来）
- 缺乏触达渠道（忘了再来）

### 症状：会员活跃度低

**诊断步骤**

1. 会员数据
   - 总会员数多少？
   - 3个月内活跃会员占比？
   - 会员平均消费频次？

2. 触达频率
   - 多久触达一次？
   - 触达方式是什么？
   - 触达内容是什么？

**常见原因**
- 会员权益无吸引力
- 触达频率太低或太高
- 触达内容无价值

---

## 四、运营类问题

### 症状：成本失控

**诊断步骤**

1. 房租占比
   - 正常水平：15%-25%
   - 超过30%需警惕

2. 人工占比
   - 正常水平：20%-30%
   - 超过35%需优化

3. 原材料占比
   - 正常水平：30%-40%（餐饮）
   - 超过50%需优化

**常见原因**
- 房租谈判不到位
- 人员配置不合理
- 供应链成本高

### 症状：人效低

**诊断步骤**

1. 人效计算
   - 月营业额 ÷ 员工数 = 人效
   - 对比同行业水平

2. 排班分析
   - 高峰期人手够吗？
   - 低峰期人手多吗？
   - 有没有闲时培训？

**常见原因**
- 排班不合理
- 员工技能不熟练
- 激励机制不到位

## 五、诊断流程图

```
用户提问
    ↓
搜知识库（IMA）
    ↓
找到相关内容？
    ├─ 是 → 提取关键信息
    └─ 否 → 联网搜索补充
              ↓
         整合信息
              ↓
         输出诊断+方案
              ↓
         加延伸阅读模块
```

## 六、快速诊断话术

**开场问诊模板**

```
老板，先别急着要方案，咱们先理理：

1. 你的店开在哪？做什么品类？
2. 现在最大的问题是什么？
3. 这个问题出现多久了？你试过什么方法？
```

**追问关键点**

- 客流问题 → 问选址、问推广
- 转化问题 → 问产品、问价格、问服务
- 复购问题 → 问会员、问触达
- 成本问题 → 问房租、问人工、问供应链

**确认问题**

```
所以你的情况是：
- [复述用户问题]
- 根本原因可能是：[给出诊断]
- 我们重点解决：[聚焦核心问题]
```
