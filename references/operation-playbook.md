## 一、获客引流

### 1.1 线下引流20招（知识库提炼）

从「线下实体爆款引流攻略」提炼，按难度分级：

**零成本（今天就做）**：
1. 门口放一块小黑板，每天写一句有意思的话
2. 给老客户发一条微信："你最近没来，想你了，送你一张券"
3. 在收银台放一个"扫码领福利"的立牌
4. 和隔壁店互相放对方的优惠券

**低成本（一周内搞定）**：
5. 9.9元引流套餐设计（价格要够低，让人不犹豫）
6. 在店门口做体验活动（试吃/试穿/试用）
7. 异业联盟：找3-5家客群相似的店，互相导流
8. 会员日/老客日：固定日期给老客专属福利

**中等成本（一个月内搭建）**：
9. 抖音团购引流套餐上线
10. 小红书发探店笔记（找本地博主合作）
11. 社群裂变：老带新送福利
12. 打卡墙设计（让顾客主动拍照发朋友圈）
13. 主题活动：节日/季节/热点相关

**需要耐心（持续做）**：
14. 本地社群运营（社区群/宝妈群/行业群）
15. 美团/大众点评店铺运营
16. 抖音短视频持续更新
17. 公众号/视频号内容输出
18. 跨行业活动联名
19. 地推+扫码+立享
20. 客户转介绍奖励体系

### 1.3 获客方案决策框架

根据用户情况推荐方案，不要一刀切：

**问用户3个问题**：
1. 你的行业是什么？
2. 你的客群画像（年龄/性别/消费水平）？
3. 你每月能拿出多少预算做获客？

**决策树**：
- 预算<500元/月 → 优先做：企微+朋友圈+社群+抖音免费内容
- 预算500-3000元/月 → 优先做：抖音团购+小红书+美团点评
- 预算3000-10000元/月 → 可以加：抖音投流+小红书博主合作+直播
- 预算>10000元/月 → 短视频矩阵+直播+全渠道铺设

## 二、私域运营

### 2.1 私域从0到1搭建SOP

从「实体门店私域运营知识地图」和「门店运营解决方案」提炼：

**第1天：装工具**
- 下载企业微信
- 完善个人信息（头像用本人照片，名字用"XX店+称呼"）
- 设置欢迎语："欢迎！我是XX店的XX，以后新品和福利第一时间告诉你~"

**第1周：加人**
- 每个进店的客人都加企微，话术："加个微信，以后有优惠我第一时间告诉你"
- 目标：第一周加50人
- 准备一个小福利（小样/优惠券/体验券）作为加微的理由

**第2周：分群**
- 把加到的人分成3层：
  - A层（月来2次以上）：核心客户群——专属福利+新品试用
  - B层（月来1次）：普通客户群——促销活动+干货分享
  - C层（还没来过/很久没来）：唤醒群——专属回归福利

**第3周：开始运营**
- 朋友圈：每天1条，内容=30%产品+40%生活/故事+30%干货
- 社群：每周2次活动（限时福利/新品预告/互动话题）
- 1对1：每月给A层客户发1条个性化消息

**第4周：看数据**
- 看这3个数字：加了多少人、社群活跃度、复购率有没有变
- 合格线：4周加200人+，A层复购率提升10%+

### 2.2 朋友圈内容规划模板

| 时间 | 发什么 | 示例 |
|------|--------|------|
| 早上8-9点 | 正能量/店铺日常 | "早！今天到了一批新货，先替你们验了✌️" |
| 中午12-13点 | 产品展示/顾客反馈 | "王姐今天又来啦，她说我们的XX用了两周效果真不错" |
| 晚上20-21点 | 干货/故事/福利 | "教你们一招：XX怎么选才不踩坑" / "今晚群里有专属秒杀，进群👇" |

### 2.3 KOC养成3步法

从「实体店经营布局私域战场」提炼：

1. **找KOC**：在老客里找那些爱分享、朋友圈活跃、社交圈广的人
2. **给权益**：给KOC专属折扣、新品试用、邀请他们参加内测
3. **促分享**：设计"分享有礼"机制——KOC分享你的内容/产品，朋友下单后双方得福利

---

### 2.4 私域运营避坑清单

| 坑 | 为什么踩 | 正确做法 |
|----|---------|---------|
| 天天发硬广 | 客户屏蔽/退群 | 7分内容3分广告，提供价值为主 |
| 所有客户一锅炖 | 不同客户需求不同 | 至少分2-3层，分层运营 |
| 社群变成死群 | 没有持续活动 | 每周至少2次互动活动 |
| 只做线上不管线下 | 私域是为线下服务的 | 线上种草→线下体验→线上复购 |
| 一上来就投流买粉 | 粉丝不精准，转化率低 | 先跑通0-1模型，再放大 |

### 3.2 SKU精简方法论

从「奶茶店3个月从亏2万到盈利15万」案例提炼：

**原则**：20%的SKU贡献80%的营收，把资源集中在头部产品。

**操作步骤**：
1. 拉出最近3个月所有SKU的销量和毛利数据
2. 按"销量×毛利"排序，找出Top 20%
3. 砍掉 Bottom 30%（销量低+毛利低的双低产品）
4. 剩余产品分3档：
   - 引流款（低价高频，吸引进店）
   - 利润款（中等价格高毛利，赚钱主力）
   - 形象款（高价低频，拉高品牌定位）
5. 每季度做一次SKU复盘

### 3.4 奶茶店扭亏复盘（实战案例）

从「奶茶店3个月从月亏2万到盈利15万」提炼核心逻辑：

**阿泽做了4件事**：
1. **思维升级**：从"卖体力"到"卖效率"，用数据代替直觉做决策
2. **SKU精简**：砍掉50%低效产品，集中推爆款
3. **私域运营**：把每个进店客人加到企微，分层运营，复购率翻倍
4. **角色跃迁**：从"自己干所有事"变成"管系统和管人"

**可复用的核心模型**：
- 用数据驱动经营效率
- 把顾客转化为可复用的数字资产
- 老板的精力从"做"转移到"管"

### 3.5 网红角落打造5步法

从「5步打造让顾客主动拍照打卡的网红角落」提炼：

1. **选主题**：找一个跟你的店调性匹配的视觉主题
2. **规划动线**：让网红角落成为进店必经之路
3. **设计钩子**：一个让顾客"忍不住拍"的视觉元素
4. **设置激励**：拍照发朋友圈/小红书，送小福利
5. **私域承接**：打卡后引导加企微，把一次性流量变成回头客
