# 平台运营指南

本手册覆盖抖音、视频号、小红书、快手等平台的实体店运营要点。当老板问"XX平台怎么做"时调用本手册。

---

## 一、抖音本地生活

### 1.1 零基础启动SOP

从「抖音本地生活商家运营指南」提炼：

**第1天：注册+认证**
- 注册抖音企业号（个体户就行，不需要公司）
- 完善店铺信息：地址、营业时间、联系方式
- 开通抖音团购功能

**第1周：发第一条视频**
- 不需要专业设备，手机+自然光就行
- 第一个视频拍什么？拍你最拿手的一个产品/服务的过程
- 文案公式："你知道XX还可以这样吗？" + 展示过程 + 引导到店

**第2-4周：持续发+看数据**
- 每周发3-5条短视频
- 重点看：播放量、点赞量、团购转化
- 播放量<500：标题/封面不够吸引，换钩子
- 播放量>1000但没转化：内容没引导到店，加团购链接和引导话术

### 1.2 短视频脚本模板

**模板1：产品展示型（最简单）**
```
开头（3秒）：一句话勾住——"你们店里XX多少钱？我这边只要XX"
中间（10-15秒）：展示产品/服务过程
结尾（5秒）：引导——"点左下角团购，只要XX元"
```

**模板2：故事型（最容易火）**
```
开头（3秒）：一个反差——"我放弃了月入3万的工作，开了这家店"
中间（15-20秒）：讲你的故事，为什么做这个店
结尾（5秒）：引导——"如果你也在XX，来找我聊聊"
```

**模板3：干货型（最稳定）**
```
开头（3秒）：一个痛点——"90%的人不知道XX怎么选"
中间（15秒）：3个实用建议
结尾（5秒）：引导——"觉得有用点个赞，更多干货关注我"
```

### 1.3 抖音获客素材SOP

从「抖音实体店获客素材SOP」提炼，5个核心标签组合：

| 标签 | 作用 | 怎么体现 |
|------|------|---------|
| 生意好 | 社会认同 | "本地客源多""每天接待XX人" |
| 专业 | 信任背书 | "十年经验""持证上岗" |
| 实在 | 拉近距离 | "先沟通再服务""不推销" |
| 口碑 | 从众心理 | "零差评""老客介绍来的" |
| 环境透明 | 打消顾虑 | 实拍店内环境，明码标价 |

### 1.4 抖音直播间搭建要点

**最低配置（0成本启动）**：
- 设备：一部手机+一个手机支架+一个环形灯（50元）
- 场景：在店里播，真实感最强
- 时间：晚上7-9点，本地人刷手机高峰

**直播内容框架**：
1. 开播前5分钟：打招呼+介绍今天直播主题
2. 中间40分钟：展示产品/服务+回答评论区问题+讲故事
3. 最后5分钟：推团购套餐+引导关注

## 二、视频号

### 2.1 视频号核心数据

从「2025视频号运营战略」提炼：

- 月活跃用户超12亿
- 商业化规模突破8000亿
- 本地生活服务增长210%
- 核心优势：微信生态内闭环（视频号→企微→小程序→成交）

### 2.2 视频号适合什么类型的店

| 适合 | 为什么 |
|------|--------|
| 中老年客群的店 | 视频号用户年龄偏大，35+为主 |
| 本地服务类 | 依托微信社交关系，同城传播强 |
| 需要信任建立的服务 | 通过朋友圈/社群导流到视频号，信任前置 |

### 2.3 公私域联动5种模式

从「5种公私域组合打法」提炼：

| 模式 | 怎么做 |
|------|--------|
| 公域引流私域 | 视频号内容引导加企微/进社群 |
| 私域打通公域订单 | 企微客户在视频号小店下单，数据打通 |
| 私域+会员体系 | 社群客户→视频号会员→专属权益 |
| 私域+视频号直播 | 社群预热→视频号直播→社群复盘 |
| 私域+视频号分销 | 老客帮你分销视频号小店商品，赚佣金 |

## 三、小红书

### 3.1 小红书核心特点

- 用户画像：18-35岁女性为主，消费力强
- 内容偏好：真实体验分享>硬广
- 转化路径：种草笔记→私信/评论→加微信→到店
- 核销率高：小红书团购核销率可达80%（是抖音的2倍）

### 3.2 小红书实体店运营3步法

**第1步：发笔记**
- 拍店里的环境、产品、服务过程
- 文案用第一人称："我发现了XX一家宝藏店"
- 标签带上城市名+品类名（如#济南养生 #同城探店）

**第2步：引导加微**
- 笔记评论区回复"私我发地址"
- 私信里发店铺位置+加微信送福利
- 小红书主页留微信号（注意平台规则）

**第3步：到店转化**
- 微信里发到店优惠
- 到店后引导拍照发小红书（给小礼物）
- 形成"小红书种草→到店体验→小红书分享"闭环

### 3.3 小红书引流注意事项

- 小红书对硬广审查严格，内容要偏"分享"而非"推销"
- 不要在笔记正文里放微信号/手机号
- 引流话术放评论区或私信
- 持续发比偶尔爆一条重要

## 四、快手

### 4.1 快手核心特点

- 用户偏三四线及以下城市
- "老铁文化"：强信任、强粘性
- 直播带货转化率高（信任经济）
- 适合：下沉市场、接地气的品类

### 4.2 快手运营要点

- 内容要真实、接地气，不要精致过头
- 多用直播，少做纯短视频
- 和粉丝互动要频繁，叫名字、拉家常
- 团购套餐价格要比抖音更低（快手用户更看重性价比）

## 五、美团/大众点评

### 5.1 基础运营清单

- [ ] 认领门店POI（如果还没有）
- [ ] 上传高质量门店照片（至少10张）
- [ ] 设置团购套餐（引流款+利润款）
- [ ] 回复所有评价（好评感谢，差评诚恳道歉+解决方案）
- [ ] 每周更新1次活动/优惠
- [ ] 关注同行数据：评分、团购量、评价数

### 5.2 团购套餐设计

| 套餐类型 | 定价策略 | 目的 |
|---------|---------|------|
| 引流款 | 9.9-29.9元 | 吸引新客进店 |
| 体验款 | 59-99元 | 让新客体验核心服务 |
| 复购款 | 199-399元/月卡 | 锁定长期消费 |
| 利润款 | 599+元/季卡/年卡 | 高毛利，核心营收 |

## 六、平台选择决策表

根据你的店的情况，帮你选最该做哪个平台：

| 你的情况 | 首选平台 | 次选 | 原因 |
|---------|---------|------|------|
| 一二线城市+年轻客群 | 抖音+小红书 | 视频号 | 年轻人在抖音和小红书 |
| 三四线/县城+中年客群 | 快手+视频号 | 抖音 | 下沉市场看快手，中年人刷视频号 |
| 餐饮/到店服务 | 抖音+美团 | 小红书 | 抖音团购+美团点评是餐饮标配 |
| 美容/养生/美业 | 小红书+抖音 | 视频号 | 视觉效果好，小红书种草力强 |
| 零售/便利店 | 美团+视频号 | 抖音 | 到店即时需求靠美团，社区靠视频号 |
| 预算有限先做一个 | 抖音 | — | 流量最大，起步门槛最低 |
