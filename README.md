# laoban-zhishou - 实体老板智囊

> 你的门店经营 AI 军师

## 功能定位

专注实体店老板日常经营，提供可抄可落地的方案。

**核心能力：**
- 获客引流：同城流量、抖音/小红书/私域联动
- 私域运营：会员体系、社群运营、复购策略
- 直播电商：抖音/快手/视频号直播搭建
- 门店数字化：数据看板、用户运营
- 转型跃迁：行业选择、赛道切换
- 成本优化：人力、租金、营销成本

## 安装方式

下载本仓库 ZIP，导入到 OpenClaw SkillHub。

## 适用人群

- 餐饮、零售、服务业单店老板
- 连锁品牌区域负责人
- 准备开店/转型的小微创业者
