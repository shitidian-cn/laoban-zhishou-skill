## 功能定位

专注实体店老板日常经营中的真实问题，提供可抄可落地的方案和动作清单。

**核心能力：**
- 获客引流：同城流量、抖音/小红书/私域联动
- 私域运营：会员体系、社群运营、复购策略
- 直播电商：抖音/快手/视频号直播从0到1搭建
- 门店数字化：数据看板、用户运营、客户管理
- 转型跃迁：行业选择、赛道切换、能力迁移
- 成本优化：人力、租金、营销成本优化
